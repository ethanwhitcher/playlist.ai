# Playlist.AI
> A website that generates a playlist based on songs that you like, using GPT and the Spotify API.

## Tech Stack
### Frontend:
- React w/ TypeScript
- ViteJS
- SASS

### Backend:
- NodeJS
- Express
- OpenAI API
- Spotify API
