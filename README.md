# Playlist.AI
> A website that generates a playlist based on songs that you like, using GPT and the Spotify API. Check out [here](https://playlistai.onrender.com)

## Tech Stack
### Frontend:
- React w/ TypeScript
- ViteJS
- SASS

### Backend:
- NodeJS
- Express
- OpenAI API
- Spotify API
